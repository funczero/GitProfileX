# FAQ

## Why is my profile not showing?

Make sure your repository name matches your GitHub username.

Example:

username/username

---

## Why are stats not loading?

GitHub Stats services may rate limit requests.

Wait a few minutes and refresh.

---

## Do I need Node.js?

Only if you want dynamic project generation.

Static templates work without it.
